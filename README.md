# UNIPULT — Универсальный веб-пульт управления роботами

**UNIPULT** — это модульный, легковесный и визуально проработанный веб-интерфейс для ручного и автоматического управления мобильными роботами. Он разработан специально для быстрого внедрения («приклеивания») в любые ваши робототехнические проекты на Python (как на чистом Python, так и в среде ROS 2 / ROS 1), запускаемые на плате **Raspberry Pi 5** под управлением Ubuntu в локальной Wi-Fi сети.

Интерфейс спроектирован по сетке **glassmorphism** (эффект матового стекла), поддерживает **две темы оформления (темную и светлую)** с плавными переходами, адаптирован для управления как с ПК (мышь/клавиатура), так и со смартфонов (сенсорный тач-интерфейс) и работает **полностью автономно (оффлайн)** без необходимости подключения к интернету.

## 🚀 Основные возможности

1.  **Ручное управление (Виртуальный джойстик):**
    *   Сенсорный джойстик на базе библиотеки `nipplejs` (работает на смартфонах и ПК).
    *   Троттлинг отправки координат (20 Гц) для плавной езды без перегрузки сети, с мгновенной отправкой сигнала остановки $(0, 0)$ при отпускании.
    *   Автоматическая стилизация джойстика под активную тему оформления с неоновым свечением.
2.  **Интерактивная карта навигации (HTML5 Canvas):**
    *   Отображение текущего положения робота в виде направленного шеврона с направлением курса ($Yaw$).
    *   **Динамическая сетка:** Масштабируется автоматически от десятков метров до **миллиметрового диапазона** ($1\text{ мм}$). Оси автоматически переводят подписи в `m`, `cm` и `mm` в зависимости от уровня приближения.
    *   **Привязка курсора (Snap-to-Grid):** Клик по карте для добавления точки автоматически притягивает её к ближайшему узлу сетки в текущем масштабе.
    *   **Реальный пройденный путь (След):** Отрисовка непрерывной неоново-зеленой траектории фактического движения робота на основе поступающей одометрии.
    *   Визуализация меток Aruco (в виде оранжевых мишеней с ID).
3.  **Адаптивные темы оформления (Темная / Светлая):**
    *   Переключение темы в один клик с помощью кнопки 🌙/☀️ в шапке панели.
    *   Выбранная тема сохраняется в `localStorage` браузера и автоматически восстанавливается при перезапуске.
    *   Все элементы интерфейса (панели, таблицы, поля ввода, кнопки, джойстик и сам холст карты) плавно переключают свои цвета с сохранением премиального эффекта матового стекла.
4.  **Улучшенное управление картой и жесты:**
    *   **Перемещение поля (Панорамирование):**
        *   *Колесико мыши (СКМ):* Зажмите колесико и тащите карту в любую сторону (блокирует стандартный скролл браузера).
        *   *Правая кнопка мыши (ПКМ):* Зажмите и перетаскивайте карту (контекстное меню на холсте отключено для удобства навигации).
        *   *Shift + Левая кнопка мыши (ЛКМ):* Зажмите `Shift` и тащите левой кнопкой — **идеально для трекпадов ноутбуков**.
        *   *Сенсорный ввод:* Просто проведите пальцем по экрану телефона для перемещения по полю.
    *   **Умное масштабирование (Zoom):**
        *   *Колесико мыши:* Прокрутка колесика вверх/вниз увеличивает/уменьшает масштаб. Алгоритм сфокусирован на курсоре — точка, на которую указывает мышь, остается на месте при зуме.
        *   *Кнопки управления:* Кнопки `🔍 +` и `🔍 -` в панели карты для быстрого зума, а также кнопка `🏠` для быстрого сброса вида к координатам $(0,0)$ и масштабу по умолчанию.
    *   **Установка меток:**
        *   *На ПК:* Клик ЛКМ (без зажатого Shift) устанавливает точку маршрута в месте клика.
        *   *На телефоне:* Быстрый тап по экрану устанавливает точку маршрута. Интерфейс распознает разницу между быстрым тапом (установка точки) и длительным движением пальца (перемещение карты).
5.  **Управление траекторией (Waypoints):**
    *   Интерактивная таблица точек с полной 6-DOF поддержкой координат ($X, Y, Z$ в метрах; $Roll, Pitch, Yaw$ в градусах).
    *   Возможность менять порядок точек кнопками ▲/▼, удалять отдельные точки или очистить весь список одной кнопкой.
    *   Импорт и экспорт маршрутов в файлы JSON.
6.  **Координаты меток Aruco:**
    *   Хранение мировых координат стационарных меток ($X, Y, Z, Roll, Pitch, Yaw$ и уникальный Tag ID).
    *   Удален выбор типа словаря для максимального упрощения структуры данных.
    *   Импорт и экспорт списка меток в JSON. Автоматическое сохранение на RPi 5 в файл `data/aruco.json`.
7.  **Безопасность и режимы:**
    *   Большая пульсирующая кнопка **E-STOP** (Экстренный останов), блокирующая любые приводы.
    *   Кнопки запуска («СТАРТ АВТО») и приостановки («СТОП АВТО») автоматического движения по траектории.
    *   Индикация заряда батареи в вольтах с автоматическим расчетом процентов для LiPo аккумулятора.

    ```bash
    python3 test_robot.py
    ```
3.  Откройте браузер на вашем компьютере:
    *   [http://localhost:8080](http://localhost:8080)

## 📱 Подключение со смартфона по Wi-Fi

1.  Убедитесь, что телефон и Raspberry Pi 5 находятся в **одной локальной Wi-Fi сети** (или подключите RPi 5 к точке доступа вашего телефона).
2.  Узнайте IP-адрес платы в терминале RPi 5:
    ```bash
    hostname -I
    ```
3.  Откройте браузер на телефоне и введите адрес вручную (обязательно с `http://`):
    *   `http://<IP_вашего_RPi5>:8080` (например, `http://192.168.1.105:8080`).
4.  **Если страница не открывается:** На Raspberry Pi 5 может быть включен файрвол. Разрешите входящие подключения командой:
    ```bash
    sudo ufw allow 8080/tcp
    ```

## 🔧 Интеграция в ваши проекты

### Шаг 1. Импорт и запуск сервера в вашем скрипте
Добавьте запуск пульта в фоновом потоке вашего основного управляющего скрипта:
```python
from server import RobotPanel

# Инициализируем панель
panel = RobotPanel(host="0.0.0.0", port=8080)

# Запускаем веб-сервер в фоновом потоке
panel.start()
```

### Шаг 2. Привязка функций-обработчиков (Callbacks)
Используйте декораторы для привязки событий с пульта к функциям вашего робота:
```python
@panel.on_joystick
def my_drive_callback(x, y):
    # y - линейная скорость (вперед/назад) от -1.0 до 1.0
    # x - угловая скорость (поворот) от -1.0 до 1.0
    set_motor_speeds(linear=y, angular=x)

@panel.on_estop
def my_emergency_callback():
    # Немедленно обесточить моторы!
    stop_all_gpios()

@panel.on_autonomous_start
def my_auto_start_callback():
    start_path_following()

@panel.on_waypoints_update
def my_waypoints_callback(wps):
    # wps - это массив точек: [{"x": 1.2, "y": 2.3, "z": 0.0, "roll": 0.0, "pitch": 0.0, "yaw": 45.0}, ...]
    update_navigation_targets(wps)
```

### Шаг 3. Отправка телеметрии на пульт
В цикле работы вашего робота (например, с частотой 10 Гц) отправляйте текущие параметры для отображения на экране смартфона/ПК:
```python
panel.send_telemetry(
    state="manual",   # Текущий режим: "idle", "manual", "auto" или "e-stop"
    battery=12.2,     # Напряжение в вольтах (проценты пульт посчитает сам)
    x=current_x,      # Мировая координата X робота (м)
    y=current_y,      # Мировая координата Y робота (м)
    z=current_z,      # Высота Z (м)
    roll=0.0,         # Крен (град)
    pitch=0.0,        # Тангаж (град)
    yaw=current_yaw   # Курс робота (град, от 0 до 360)
)
```

*Для интеграции с помощью нейросети просто скопируйте и отправьте файл **`AI_README.md`** вашему ИИ-помощнику вместе с кодом вашего робота.*
